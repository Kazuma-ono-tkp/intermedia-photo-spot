# Intermedia Photo Spot MVP v40 - QR JPEGアップロード版

## v40で修正したこと
- QR共有用アップロード画像をPNGからJPEGへ変更
- JPEG品質 0.86 でBlobアップロード
- アップロード中にファイルサイズを表示
- API側の上限を8MBから15MBへ拡張
- 成功時は「QRでスマホ保存できます」と「画像を開く」リンクを表示
- 画面表示・写真保存用の元画像は維持

## 狙い
高画質PNGが大きすぎてBlobアップロードに失敗するケースを避けます。

## テスト
1. 撮影
2. 撮影完了画面で「QR用画像をアップロード中... ○MB」と表示されるか確認
3. 「QRでスマホ保存できます」が出るか確認
4. 「画像を開く」でBlob画像が開くか確認
