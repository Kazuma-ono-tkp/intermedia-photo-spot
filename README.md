# Intermedia Photo Spot MVP v41 - QR画像表示版

## v41で修正したこと
- QRコードが表示されない問題を修正
- canvas描画ではなくQR画像として表示する方式へ変更
- Blob URL発行後、QR画像を直接表示
- 「画像を開く」リンクは継続

## 補足
テスト段階向けに無料QR画像生成サービスを使用しています。
本番版では自前QR生成ライブラリを同梱する方式に切り替えるのがおすすめです。

## テスト
1. 撮影
2. 「QRでスマホ保存できます」が表示される
3. 左側にQR画像が表示される
4. QRをスマホで読み取り、画像URLが開く
